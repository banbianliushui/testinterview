宽度高度百分比

块元素: 宽高分别相对于包含块的百分比，是盒模型中content-box内容区宽高来计算百分比，即如果是border-box 内部宽度的百分比计算值 = (外部width - (padding + border)) * a%
行内元素-非替换： 行内元素随内容自动扩大width，设置宽度无效，设置line-height有效

padding百分比

上下左右内边距都是相对于包含块的宽度计算

块元素: 盒模型中content-box内容区宽高来计算百分比，即如果是border-box 内部宽度的百分比计算值 = (外部width - (外部padding + 外部border)) * a%；如果是content-box就是父容器的width为参考。

行内元素-非替换： 左右padding有效果（布局效果），上下padding只有显示效果，不影响其他行的布局。

top,left,right,bottom百分比

CSS框模型：左外边界-> 左外边距-> 左边框->左内边距->左内边界 ->内容区->右内边距->....

下面说的左边界指左边框的右边

position:absolute,,移动在边框内移动，计算百分比的参照也是边界内的容器宽高(padding+contentheight)

position:relative,计算百分比的参照是content内容区宽高,相对于它自身所在位置移动

相对包含块最近边的偏移，top,bottom相对于包含块的高度(padding+contentheight)计算,left,right相对于包含块的宽度（padding+content宽度，和上面不同，待找依据）计算

left描述了定位元素的左外边距（margin-left）边界在其包含块左边界（border-left）右边（正值）有多远

注： 测试chrome left:0,top:0;是相对于包含块border内部开始计算的，如果为负数就覆盖border
